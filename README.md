# 🚗 Car Price Dataset Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a car price dataset using Python. The goal is to identify patterns and relationships between different features such as mileage, brand, and fuel type, and how they influence car prices.

## 📊 Dataset Description

The dataset is synthetically generated and contains the following features:

* **Brand** – Car manufacturer (Toyota, Honda, BMW, etc.)
* **Mileage** – Distance covered by the car (km/l)
* **Fuel_Type** – Type of fuel used (Petrol, Diesel, Electric)
* **Price** – Estimated price of the car

## 🔍 Analysis Performed

### 1. Price vs Mileage

* Scatter plot used to visualize the relationship
* Helps understand how mileage affects car price

### 2. Brand-wise Price Comparison

* Average price calculated for each brand
* Bar chart used for comparison

### 3. Fuel Type Trends

* Analyzes how fuel type impacts pricing
* Shows average price distribution across fuel types

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* NumPy

## ▶️ How to Run

1. Open Google Colab
2. Copy and paste the Python script
3. Run all cells
4. Visualizations will be displayed automatically

## 📈 Output

* Scatter plot: Price vs Mileage
* Bar chart: Brand-wise average price
* Bar chart: Fuel type trends

## 📌 Conclusion

* Higher mileage generally reduces car price
* Premium brands like BMW and Audi have higher average prices
* Electric cars tend to have higher pricing compared to petrol and diesel vehicles
